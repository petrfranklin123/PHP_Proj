Нужно написать одностраничное приложение с учетом всех условий и требований. Примерный макет design.png

1. На странице отображается таблица игроков из файла players.json. Высота таблицы фиксированная. Список игроков должен прокручиваться, при этом заголовок таблицы должен оставаться на месте. Колонки: ID, Имя, Уровень, Онлайн.

2. Нужно реализовать сортировку в таблице, фильтрацию по онлайн и поиск по имени.

3. Если нажать на игрока в списке, то под ним должен появиться блок с действиями. При изменении размеров окна блок должен находиться под выбраным игроком на том же месте. Кнопки действий: "Показать профиль" и "Скрыть игрока".

Если нажата кнопка "Показать профиль", то должен появиться попап с информацией об игроке.
Если нажата кнопка "Скрыть игрока", то игрок не должен отображаться в списке, показать всех игроков можно нажав кнопку "Показать всех".

Условия: 

	javascript

	React, Less, Webpack + Babel, для сборки можно использовать любые webpack плагины и лоадеры. Другие библеотеки использовать запрещено.

	Браузеры: Chrome, IE 11

	Приложение должно собираться командой npm run build в директорию release.
